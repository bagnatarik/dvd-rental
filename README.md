# Analyse des locations et revenus: DVDRental

## Contexte

Analyse exploratoire des locations et revenus d'une entreprise de location de film sur la période de Février 2007 à Mai 2007 à partir de la base de données DVDRental.

**Jeu de données utilisé :** [DVDRental Dataset](https://neon.com/postgresqltutorial/dvdrental.zip)
**Outils utilisés :** PostgreSQL

---

## Questions analysées

- Quels sont les meilleurs clients de l'entreprise ?
- Quel est l'écart entre le client ayant le plus de location et le client ayant le moins de location ?
- Quelle est la durée moyenne de location par film ?
- Quel est le chiffre d'affaires par mois ? Il y a t'il une différence significative entre deux mois ?

---

## Résultats clés

- Le Top 5 présente des volumes homogènes sans écart significatif entre eux.
- Aucun film ne se démarque donc significativement.
- Le groupe des films les plus rentable est homogène sans écart significatif.
- L'écart entre les revenenus généré par les staff est de 2,6% et est non significatif.
